GG3D package
==============

GG3D v6.0 is a program library for carrying out forward modelling and inversion of surface, borehole, and airborne gravity gradiometry data in 3D. The data can be full-tensor gradiometer (FTG) or the raw components of the Falcon or VK1 systems. The version number is set to be consistent with latest version of the potential-field inversion codes. The contents of this manual are as follows:
